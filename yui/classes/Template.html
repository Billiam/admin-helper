    Render template string with provided arguments

    Variables are wrapped in single braces

    Parameters:

    • template String

      Template

    • data Object | Array

      Template variables

    • raw Boolean

      If true, input will not be HTML escaped

    Returns:

    String:

    Example:

    var html = Template.render(
    '<b>{name}</b> - {title}',
    {
        name: 'Rudy',
        title: 'Sysop'
    }
);
console.log(html);
//> <b>Rudy</b> - Sysop
